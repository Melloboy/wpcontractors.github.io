PHP Developer - Custom E-commerce Gateway (Part-time)

Remote

Looking for:

Someone with e-commerce experience building a gateway with Authorize.net or something similar like WorldPay calling hash function from PHP….pulling information from URL then hash it and managing the hash from PHP.

This person would be aiding any merchants that have problems integrating with the payment gateway. So requires someone with excellent communication skills to communicate with the internal technical team and the 90+ merchants. A troubleshooter, someone that can think outside of the box and solve problems.

Project:

The college of engineering is building a payment gateway. There will be two, one with Worldpay which someone on campus is building now so this person would interface with them and then they will work on building the “Authorize.net” payment gateway. So there 92 merchants that will be using this gateway for payments. So this routes traffic from campus e-commerce sites.

1. Build the gateway

2. Assist merchants with issues connecting to the gateway

3. The manager is writing the PCI documentation but wants this resource to proof it to make sure it is correct

So with e-commerce sites comes hashing for integrating with the payment gateway. This resource must be strong with PHP, JavaScript and hashing for e-commerce sites.

Details:

  • This person can work 100% remote, even after the current pandemic
  • The contract starts with 6 months but will more than likely get extended
